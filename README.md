# Keycard-Reminder
A motion activated LCD reminder that detects movement using a PIR sensor and displays a reminder message on an I2C LCD screen.

# Purpose
I live in a dorm, and I often forget to take my keycard before leaving. Due to this, I have been locked out of my room multiple times. Therefore, I made a sensor that reminds me to take my keycard with me before leaving the room!!
I also added a buzzer to alert me just in case I do not spot the screen light up.

# Electric components 
I used the following components for this project:
1) PIR Sensor
2) I2C LCD
3) ESP32 Microcontroller
4) Transistor
5) 1 kΩ resistor
6) Active buzzer
7) jumper wires
8) Breadboard
